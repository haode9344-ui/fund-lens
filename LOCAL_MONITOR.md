# 电脑本地监控版

如果要做财联社/新浪 7x24、公司公告监控，建议用电脑本地版或云端版。GitHub Pages 只能前台打开页面，不能在后台定时运行。

## 启动

双击：

```text
start-fund-lens.bat
```

或在命令行运行：

```powershell
python app.py
```

打开：

```text
http://127.0.0.1:8765
```

## 当前监控内容

- 新浪财经 7x24 快讯。
- 东方财富重仓股公告。
- 按基金名称、前十大重仓股、行业关键词、正负面词做关联。
- 页面会显示“新闻舆情与公告预警”。

## 限制

- 电脑关机后监控停止。
- 本地 HTTP 不能稳定给 iPhone 做后台推送；手机需要打开页面查看预警。
- 新闻情绪和实际涨跌不一定一致，系统会提示“等待行情验证”。
