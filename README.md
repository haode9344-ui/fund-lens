# Fund Lens

一个给支付宝基金做短期辅助分析的本地网页应用。它输入 6 位基金代码，抓取东方财富/天天基金公开净值走势、最新季报重仓股、重仓股实时行情和行业信息，用近 5/10/20 日动量、30 日波动率、90 日回撤和关联持仓影响估算未来约 5 个交易日的涨跌方向、涨跌幅区间和净值区间。

## 运行

```powershell
python app.py
```

打开：

```text
http://127.0.0.1:8765
```

Windows 上也可以直接双击：

```text
start-fund-lens.bat
```

## 使用

1. 在支付宝基金详情页找到 6 位基金代码。
2. 输入代码并点击“分析”。
3. 查看偏涨/偏跌、概率、估算涨跌幅区间、估算净值区间、近 5/10/20 日表现和净值曲线。
4. 查看重仓关联影响，包括前十大持仓、行业集中度、今日涨跌和对基金净值的估算贡献。
5. 展开近 5/10/20 日明细，查看准确日期和每日涨跌。
6. 查看明天涨跌详细判断、前几日复盘原因、市场环境、影响事件和买进理由。
7. 电脑本地部署版会额外显示新闻舆情与公告预警，抓取新浪财经 7x24 快讯和东方财富重仓股公告。

## 苹果设备

- iPhone Safari 打开服务地址后，可以用“分享”->“添加到主屏幕”安装为 PWA。
- 只用 GitHub Pages 也可以运行基础分析，打开仓库的 Pages 地址即可。
- `ios/` 目录提供 SwiftUI 自签模板。Windows 不能生成 IPA，拿到 Mac 上用 Xcode 新建 iOS App 后替换源码即可签名安装。
- 如果要电脑关机后手机也能用，需要部署到云端。见 `DEPLOY.md`。
- 如果要财联社/新浪 7x24、公司公告监控和预警，建议用电脑本地版或云端版；GitHub Pages 不能后台定时监控。
- 电脑本地监控版说明见 `LOCAL_MONITOR.md`。

## 说明

这个工具只基于历史净值做统计估算，不能保证未来涨跌，也不构成投资建议。基金受市场、仓位、行业、汇率、政策、基金经理调仓等多因素影响，模型结果只能当作决策前的一个参考面板。
